## 📌 Sobre o Projeto

O novo site do **Cubo Mágico Hub** foi desenvolvido com o paradigma de "Design Engineering", visando uma Experiência de Usuário (UX) premium. A interface foi planejada para transmitir calma, seriedade e acolhimento através de cores pastéis, animações suaves, cantos arredondados modernos e micro-interações físicas responsivas.

## 🚀 Principais Features

- **Arquitetura Mobile-First:** Perfeitamente otimizado para dispositivos móveis com suporte a gestos de deslizar (swipe) em carrosséis nativos de alta performance.
- **Micro-Interações Físicas:** Elementos reagem ao toque do usuário (botões e botões encolhem sutilmente ao serem clicados `transform: scale(0.97)`), proporcionando sensação tátil.
- **Filtragem de Serviços Segmentada:** Navegação dinâmica entre áreas de cuidado (Saúde Mental, Corpo, Estética, Integrativo) usando hooks em React.
- **Chamadas de Ação (CTA) embutidas:** Todos os contatos fluem diretamente para o WhatsApp de forma transparente, carregando mensagens personalizadas.
- **Estilização com CSS Modules:** CSS puro, escalável, livre de conflitos e escrito manualmente, não sobrecarregando o pacote da aplicação da web.

## 🛠️ Tecnologias Utilizadas

- [React 19](https://react.dev/) - Biblioteca para construção de interfaces.
- [Vite 8](https://vitejs.dev/) - Bundler de build focado em velocidade extrema.
- [TypeScript](https://www.typescriptlang.org/) - Superset de JavaScript com tipagem estática.
- [Lucide React](https://lucide.dev/) - Coleção de ícones vetoriais modernos.
- CSS Modules - Estilização limpa baseada em módulos locais.
